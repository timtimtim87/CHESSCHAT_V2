# Portfolio Diary

Purpose: day-by-day engineering diary for CHESSCHAT portfolio build sessions.

How to use:
- Keep one file per day: `YYYY-MM-DD.md`.
- Record three things each session:
  - What we built
  - How we built it
  - Why we made those choices (tradeoffs and thought process)

Primary audience:
- Tim (portfolio/interview prep)
- Future AI agents continuing this project
